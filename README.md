# Магазин тортов — Telegram-бот

Бот для продажи тортов через Telegram.

## Установка

1. Установите `uv` (менеджер пакетов Python)
2. Установите зависимости:
   ```bash
   make install
   ```

## Настройка

Установите переменные окружения:
- `BOT_TOKEN` — токен Telegram бота (обязательно)
- `PAYMENT_PROVIDER_TOKEN` — токен провайдера платежей (опционально)

## Запуск

```bash
make run
```

Или напрямую:
```bash
python -m bot.app
```

## Тестирование

```bash
make test
```
